# Jordan Gould | Portfolio

Welcome to my professional portfolio site, hosted via GitHub Pages.

## 🚀 Overview
This site showcases my journey as an MSIT student at Florida State University, my experience in IT Analysis, and my passion for leadership in technology.

## 🎨 Design Concept
- **Vibe:** Minimalist, professional, and accessible.
- **Color Palette:** Clean whites, soft greys, and FSU-inspired accents.
- **Framework:** Vanilla HTML5 and CSS3.

## 📁 Repository Structure
- `index.html`: The core structure and content of the site.
- `style.css`: Custom styling and layout definitions.
- `assets/`: Folder for professional headshots and project images.
